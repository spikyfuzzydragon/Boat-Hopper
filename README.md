# Boat Hopper

<img width="928" height="463" alt="image" src="https://github.com/user-attachments/assets/b191bcd2-fb40-41a3-8670-8e1f324c70d5" />

This is a game called Boat Hopper, which I made for the [Game-off 2025 game jam on itch](https://itch.io/jam/game-off-2025).

You are always moving. You can turn around when you press x or hit a wall. Press z to jump.

The goal is to get to the end - good luck!
